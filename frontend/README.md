# Frontend - Hospital Chatbot Interface

Giao diện HTML để test chatbot.

## Cách sử dụng

### Cách 1: Mở trực tiếp file HTML
1. Đảm bảo backend server đang chạy tại `http://localhost:8000`
2. Mở file `frontend/index.html` bằng trình duyệt (double-click hoặc drag vào browser)

### Cách 2: Truy cập qua server
1. Chạy backend server: `python run.py`
2. Mở trình duyệt và truy cập: `http://localhost:8000/`

## Tính năng

- ✅ Giao diện đẹp, hiện đại
- ✅ Chat real-time với chatbot
- ✅ Hiển thị gợi ý câu hỏi tiếp theo
- ✅ Lưu lịch sử cuộc trò chuyện
- ✅ Hiển thị trạng thái kết nối
- ✅ Responsive design
- ✅ Typing indicator khi chatbot đang trả lời

## Lưu ý

- Đảm bảo backend server đang chạy trước khi sử dụng
- Nếu gặp lỗi CORS, kiểm tra lại cấu hình CORS trong `backend/main.py`
- Patient ID là optional, có thể để trống
